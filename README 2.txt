To have this project started up. You need to have following installed:
1) Mongo DB
2) Node JS

To get started with the project:
1) Copy and paste the project directory
2) Start the mongo server 
3) Import the mongo scripts into the Mongo server to create predefined collections
4) Go to Node project directory (it already have required dependecies and required mongo connection)
5) Type: node ecomserver.js

server should start and accessible at http://localhost:8081/
To browse the project please use a Web browser
